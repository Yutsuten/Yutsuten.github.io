# Javascript: Array

### Check if contains value
#### index.js
```javascript
var myArray = [2, 3, 5]
console.log(myArray.indexOf(3) != -1)
```

### Map

#### index.js
```javascript
var list = [2, 3, 5, 7];
var doubles = list.map(num => num * 2);
// [4, 6, 10, 14]
```

### Simple methods
```javascript
// Join values into a string
var list = ['car', 'bike', 'cookie'];
list.join(' and ');
// Returs: car and bike and cookie
```
