## Syntax

```shell
less OPTIONS FILE
-n  # Supress line numbers
-S  # Chop long lines
-X  # Do not clean the screen
-F  # Quit if one screen
-R  # Show colors
+   # Automatically run commands
```

## Commands

```shell
g  # Go to first line
G  # Go to last line
F  # Follow mode - <Ctrl + C> to cancel
```
