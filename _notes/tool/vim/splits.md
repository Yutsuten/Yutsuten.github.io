### Splitting
```shell
<c-w>s  # Split horizontally
:sp
<c-w>v  # Split vertically
:vs
```

### Rotating
```shell
<c-w><c-r>  # Rotate (invert) windows
```

### Split size
```shell
# Change size horizontally
<c-w> +
<c-w> -
<c-w> 10 +
<c-w> 10 -

# Change size vertically
<c-w> >
<c-w> <
<c-w> 10 >
<c-w> 10 <

# Equal dimensions
<c-w> =
```
