### Buffer list

```vim
:buffers
:ls
```

### Change buffer

When typing the name of a buffer, `<Tab>` can be used for auto-completion.

```vim
:b <num>
:b <name>
```

### Delete buffer

When typing the name of a buffer, `Ctrl-A` can be used to target multiple buffers.

```vim
:bd <num>
:bd <name>
```
