For more details, check the help for `list`.

## Basic usage

```vim
let mylist = [1, 'two', 3, 'four']
let numbers = range(5)  " => [0, 1, 2, 3, 4]"
mylist[1]
```

## List operations

```vim
let mylist = [1, 2] + [3, 4]  " => [1, 2, 3, 4]"
```

## List functions

```vim
empty(list)
len(list)
max(list)
min(list)
```
