## MySQL

```sql
CREATE DATABASE dbname;
DROP DATABASE dbname;
USE dbname;
SHOW DATABASES;
SHOW DATABASES LIKE '%name%';
```

## PostgreSQL

```sql
CREATE DATABASE dbname;
DROP DATABASE dbname;
\c mydb   -- \connect to database
\l        -- \list databases
\q        -- \quit
```
